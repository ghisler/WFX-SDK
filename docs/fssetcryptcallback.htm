FsSetCryptCallback
FsSetCryptCallback is called when loading the plugin. The passed values should be stored in the plugin for later use. This function is only needed if you want to use the secure password store in Total Commander.
 
Declaration:
 
void __stdcall FsSetCryptCallback(tCryptProc pCryptProc,int CryptoNr,int Flags);
 
Description of parameters:
 
pCryptProc Pointer to the crypto callback function. See CryptProc for a description of this function
 
CryptoNr A parameter which needs to be passed to the callback function
 
Flags Flags regarding the crypto connection. Currently only FS_CRYPTOPT_MASTERPASS_SET is defined. It is set when the user has defined a master password.
 
Return value:
 
This function does not return any value.
 
Remarks:
 
You can use this callback function to store passwords in Total Commander's secure password store. The user will be asked for the master password automatically.
